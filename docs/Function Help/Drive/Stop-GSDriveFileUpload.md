# Stop-GSDriveFileUpload

## SYNOPSIS
Stops all Drive file uploads in progress and disposes of all streams.

## SYNTAX

```
Stop-GSDriveFileUpload [<CommonParameters>]
```

## DESCRIPTION
Stops all Drive file uploads in progress and disposes of all streams.

## EXAMPLES

### EXAMPLE 1
```
Stop-GSDriveFileUpload
```

Stops all Drive file uploads in progress and disposes of all streams.
